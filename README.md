# Cyclone V Nios II
Nios II RISC-V embedded processor implementation for DE0-CV (Cyclone V)
## Used IPs
- Nios II Processor
- Clock Bridge
- Reset Bridge
- On Chip Memory Intel FPGA IP
- JTAG UART Intel FPGA IP
- PIO (Parallel I/O) Intel FPGA IP
